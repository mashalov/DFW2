Raiden EMS 1.0.1.122 - 07.04.23

1. 	В автоматике и сценарии эмулируется "кумулятивное" изменение параметров модели в момент времени по формулам, которые являются функциями от текущего значения V. D RUSTab этот эффект был получен без определенных намерений и в данный момент используется в сценнариях СМЗУ. И хотя этот эффект, строго говоря, является некорректным поведением, он учитывается для совместимости с RUSTab. Включение эмуляции доступно с помощью параметра "ChangeActionsAreCumulative". По умолчанию эмуляция включена.
2.	Нагрузки узлов для автоматики/сценария вычисляются на каждом шаге интегрирования с учетом ДСХН. Ранее нагрузки обновлялись только в моменты дискретных изменений.
3.	Введена модель СМ 4-го порядка (двухконтурный Парк в форме ЭДС)
4.	Введено инерционно-форсирующее звено.
5.	Постоянные элементы матрицы Якоби в большинтсве моделей не рассчитываются заново при обновлении матрицы.

Raiden EMS 1.0.1.121 - 20.02.23

1. 	Количество файлов результатов в каталоге может быть ограничено с помощью соответствующего параметра RUSTab. Количество файлов поддерживается на заданном уровне за счет удаления наиболее старых файлов.

Raiden EMS 1.0.1.120 - 17.02.23

1. 	Появилась возможность изменять параметры расчета с помощью файла config.json в рабочем каталоге. Если файл существует и соответствует формату - параметры из него перекрывают те, которые были заданы в модели.
2.  Добавлена модель глобального для всей модели "автомата безопасности турбин". При превышении заданной частоты вращения любого генератора он будет отключен от сети и исключен из модели. Уставка частоты по умолчанию +12% номинала.

Raiden EMS 1.0.1.119 - 14.02.23

1. 	Компилятор поддерживает нестандартные пусковые органы с произвольными формулами, не связанными с адресованным объектом расчетной модели.
2.	Исправлена ошибка, приводившая к отказу при адресации узла, включенного в суперузел.

Raiden EMS 1.0.1.118 - 03.02.23

1. 	Добавлена компиляция сценария из шаблона *.scn RastrWin. Сценарий формирует отдельный модуль, независимый от модуля автоматики и работает параллельно с ней.
2.	Для всех компилируемых пользовательских моделей в рабочем каталоге Raiden введен общий подкаталог - Factory.
3.	В имени инсталлера введена версия пакета.

Raiden EMS 1.0.1.117 - 01.02.23

1. 	Введена система кэширования исполняемых модулей скомпилированных моделей. Готовый модуль определяется по хэшу исходного текста пользовательского устройства 
	и проверяется на соответствие данному тексту.
2. 	Добавлена автоматическая очистка каталога, в котором кэшируются скомпилированные модули пользовательских моделей. Поддерживаются ограничения по количеству 
	файлов в кэше и по размеру кэша. Предусмотрено ограничение частоты очистки кэша.
